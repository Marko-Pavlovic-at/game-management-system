# Game Management System

A personal game library app — built as a hands-on way to learn React.

## What it does (MVP)

- **See your games** as a grid of cards (title, genre, rating).
- **Add a game** with a form, and watch it appear in the grid instantly.
- **Search / filter** the list as you type.
- **Persist** your games — they survive a browser refresh (saved locally).
- **Real games** pulled from a public games API (RAWG), not just made-up data.
- **Game detail pages** — click a game to open its own page (its own URL).

## Post-MVP (the "completion sense" features)

These come after the MVP is working:

- **Play timer** — press "Play" on a game to start a timer, so you can see how
  much time you've spent on each game (total playtime per game).
- **Goals & notes per game** — each game has a little notes/goals list where you
  can write down goals (e.g. "beat the final boss", "100% the map") and check
  them off, giving every game a real sense of progress and completion.
